
To make a single folder use the command `mkdir` or `md` and then the folder name:

```
mkdir foldername
```

<br>

To create a folder in another folder  just cd to it and enter the `mkdir` and the name of the folder.

You can also make a subfolder using mkdir command itself if you have a folder named  "Photos" and you wanted to create  a folder named "Odis" use the following command:

```
mkdir Photos\Odis
```

<br>

To make several folders at once, add the folder names to the command:

```
mkdir Documents Photos Videos
```  

<br>

To create a file that is empty that you can use with the appropriate apps is  as follows:

```
type nul > name.ext
```

You can also create an empty file with the echo command as follows:

```
echo. > example.txt
```

To create a file with text inside use the echo command with desired text and then the file name:

```
echo Welcome to How-To Geek > howtogeek.txt
```
