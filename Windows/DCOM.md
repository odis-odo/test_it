- DCOM (Distributed Component Object Model) Microsoft technology 
- Allows software components on different networks to communicate with each other
- Extension of the COM(Component Object Model) 
- Used in Windows environments for various services and applications  


#### Common DCOM Errors   

##### DCOM Error 1115  

- Less common can indicate issues with Windows Update and other system services  

+  Reinstall or reset the service: Sometimes reinstalling the affected service or resetting it can resolve the issue.  
+ Reinstall or Reset the App: Reinstalling the affected application or resetting it can sometimes fix the issue.   


##### DCOM Error 10016   

- This error is related to permissions and can be quite common.   

+ Permissions Fix: Ensure that the user or service has the necessary permissions to access DCOM
+ Component Services Tool: Use the Component Services administrative tool to configure DCOM settings
