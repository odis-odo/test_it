#### Convert audio into ACC  

- open audio file or drag it into file explorer into itunes and it should appear in your library
- Right-click the file and click "song info" you will see a pair of fields labelled  "start" and "stop"
- select the audio track and expand the "file" menu
- hover over the "convert" submenu and  select " Create AAC Version"
- there  should be two copies the original file and the AAC file  


#### Turn AAC File into M4R Ringtone  

- right click one of the heading in itunes library  and enable "kind" column
- a new "kind" column will appear telling you which file type is which.
- right click the AAC file in itunes and click "show in Windows Explorer"
- a file explorer window with the audio file will appear
- change the extension from M4A to M4R (song.m4a to song.m4r)  


#### Add the Ringtone File to iPhone  

- connect iPhone to pc  with usb cable  and unlock the phone and tap "trust"
- phone will appear in itunes sidebar then click "tones" button in sidebar
- open file explorer window and drag the M4R ringtone into the itunes tones library
- click the device icon and then press "sync" button at the bottom the itunes app to transfer to iphone   


#### Assign the Ringtone to All incoming calls  


- Disconnect the phone from PC open settings app scroll to "sound" and tap  "ringtone"
- the ringtone will appear at the top of the list and select it to assign the ringtone to all incoming calls.
