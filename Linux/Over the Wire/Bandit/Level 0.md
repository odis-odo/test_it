[Bandit](https://overthewire.org/wargames/bandit/)

For this level just log in  using ssh on port 2220 and connecting with the host name and user  name which is the same.

- `username: bandit0`  
- `password: bandit0`


To ssh into the game the command id as follows:

~~~
ssh -p 2220 bandit0@bandit.labs.overthewire.org
~~~

Then just enter the password when prompted.
