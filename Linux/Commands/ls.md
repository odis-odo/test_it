### ls - list directory contents

<br>

  Basic usage of the ls command will display contents of he directory:

	ls

<br>

  Use the -l to give a long listing of files in a directory:

	ls -l

<br>

  Use the -h shows human readble file sizes in  mb/gb etc:

	ls  -h

<br>

  Use the -a to view hidden files in a directory:

	ls -a

<br>
  
  Use the --color option to colorize the output if it is not shown:

	ls --color

<br>
  
  Use the -F will show a trailing / for directories and files will not;

	ls -F

<br>

  The -t option sorts the output by time instead of alpahbetical:

	ls -t

<br>

  All of these options can be combined in some way.
