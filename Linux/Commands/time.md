basic usage of the time command to how long it takes to run a command:

	time ls

cat.txt   echo.txt      locate.txt  passwd.txt  watch.txt
date.txt  hostname.txt  nmap.txt    ss.txt

real	0m0.001s  
user	0m0.001s  
sys	0m0.000s  

real refers to real time it takes to run the command.

user mode user side of things like running firefox

sys mode is the kernel side of things
