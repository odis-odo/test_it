### top - display Linux processes

<br>

  Basic usage of the top command:

	top

<br>

  Shift + V shows the child process of a prcess that has been run in the command column.
  
  Shift + p (CAp P) sorts the process table by cpu usage.

  Shift + m (Cap M) sorts the process table by memory usage.

  Press a will bring up a pid to kill a process.

  Press d will change change the delay of the process table updates.
