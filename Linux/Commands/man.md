### 	man - an interface to the system reference manuals

<br>

  Pressing h while in the man page will show the shortcuts to navigate thought the page.

  Using the / searches for keywords inn the man page and pressing n will move to the 
  next occurrence of the keyword.


  Pressing g will bring you back to the beginning of the man page and pressing shift + g 
  will go to the end of the page.

<br>

  To navigate to specific  sections of the man page just put the number after the command:

	man 5 passwd
  
  This will display the section when entered instead of the default 1.

<br>
  
  The sectons of the man page:

  Section 1 : Command info  
  Section 2 : System calls  
  Section 3 : Library calls  
  Section 4 : Special files  
  Section 5 : File formats and conventions  
  Section 6 : Games  
  Section 7 : Misc  
  Section 8 : System administrations commands  
  Section 9 : Kernal routines  

  Not all commands will have all these sections.

<br>

  To view the man page for a builtin command:

	man builtins
