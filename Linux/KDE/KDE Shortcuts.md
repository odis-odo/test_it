### Shortcuts for desktop management  

- `Meta + W` opens the desktop overview where you can add new desktops or switch between them quickly.
+ `Meta + G` this displays desktops in a gridview where you switch between them or move windows around.
- `Ctrl + f1,f2,f3` to switch to different desktops each corresponds to a desktop number.
+ `Ctrl + F9`  this will show all the windows from the current desktop
- `Ctrl + F10`  this will show the windows open across all desktops  

&nbsp;

### Shortcuts for a better workflow  

- `Ctrl + T` this will open a new tab in dolphin move file between folders.
+ `Ctrl + F7`  will all open windows of the application  

&nbsp;

### Shortcuts for terminal  

- `Ctrl + alt + T`  this will open the terminal
+ `Ctrl + A`  will bring the cursor to the beginning of line 
- `Ctrl + E`  will bring the cursor to the end of the line
+ `Ctrl + shift + W`  will close terminals open in split view  

&nbsp; 

### Shortcuts for krunner  

- `Alt + Space` to launch krunner


&nbsp;


### Shortcuts that are windows like  

- `Win + .` this will open the emoji window
+ `Win + V` this will open clipboard history
- `Win + E`  will open the file manager
+ `Win + I`  this will open the system settings but may have to be configured
- `Win + D`  this will minimize all windows on the desktop will put everything back and pressed again.  

&nbsp;

### Cool shortcuts to enhance workflow   

- `Meta + F6` this will bring your mouse cursor to the center of the screen
+ Shaking your mouse will make your cursor will enlarge temporarily until you stop
- `Meta + =`  this will zoom into your screen as much as you need.
+ `Meta + -`  will zoom out
- `Meta + 0`  will return the screen to default
